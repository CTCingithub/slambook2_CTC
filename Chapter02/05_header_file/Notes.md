# 使用头文件

头文件通常包含函数声明、宏定义、类型定义等，它们可以被多个源文件引用。使用头文件可以提高代码的可读性和可维护性，同时也可以避免重复定义。

头文件的使用方法如下：

1. 建立`include`文件夹，所有定义的头文件都放在这个文件夹中;
2. 建立`src`文件夹，所有源文件都放在这个文件夹中;
3. 在头文件中，使用`#ifndef`和`#endif`循环，以及`#define`命令，防止重复引用头文件引起重定义错误;
4. 在`CMakeList.txt`中，添加`include_directories`命令，将`include`文件夹添加到编译器的搜索路径中;
5. 在`CMakeList.txt`中，添加生成可执行程序的命令`add_executable`，以及链接库的命令`target_link_libraries`。
